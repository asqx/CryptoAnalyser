This file describes mode.
1. Encrypt. It this mode, the file is encrypted using a key that you specify yourself.
2. Decrypt. Similarly, as with encrypt. It only decrypts the file.
3. Bruteforce. This mode brute force picks up all possible keys and decrypts filr.
4. Analyse. The mode contains statistics on the most used russian letters. It does an analysis of the encrypted file, then makes a selection of letters and replaces them.
5. Exit. To exit the program - select this mode.
A large text is needed, decoding may not be entirely succesfully. It all depends on the frequency of characters in the text.
